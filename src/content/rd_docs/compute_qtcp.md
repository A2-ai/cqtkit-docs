## Description

Computes Population corrected QT

## Usage

```r
compute_qtcp(
  data,
  use_BL = TRUE
)
```

## Arguments

| Name | Description |
|------|-------------|
| `data` | Data frame containing QT data |
| `use_BL` | a bool to use BL for correction or not. |

## Returns

a dataframe with QTCP added in columns

## Examples

```r
compute_qtcp(data)
```
