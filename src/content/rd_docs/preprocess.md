## Description

Pre-processes data
 Computes QTcB, QTcF, deltaQTcF, deltaQTcB, deltaHR, deltaQTcB Baseline Mean,
 deltaQTcF Baseline Mean, deltaHR Baseline Mean

## Usage

```r
preprocess(
  data
)
```

## Arguments

| Name | Description |
|------|-------------|
| `data` | Data frame containing QTc data |

## Returns

dataframe with deltas computed from BL

## Examples

```r
preprocess(data)
```
