## Description

Computes QTCF from QT and RR for both Baseline and non-baseline measurements

## Usage

```r
compute_qtcf(
  data
)
```

## Arguments

| Name | Description |
|------|-------------|
| `data` | a dataframe containing QT, RR, QTBL, RRBL |

## Returns

a dataframe which contains x, plus QTCF and QTCFBL

## Examples

```r
compute_qtcf(data)
```
