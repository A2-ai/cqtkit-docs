## Keyword

internal

## Description

See  magrittr:: %>%  for details.

## Usage

```r
lhs %>% rhs
```

## Arguments

| Name | Description |
|------|-------------|
| `lhs` | A value or the magrittr placeholder. |
| `rhs` | A function call using the magrittr semantics. |

## Returns

The result of calling  rhs(lhs) .
