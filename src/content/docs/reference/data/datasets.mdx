# `cqtkit` Included Datasets

`cqtkit` has datasets from four different drugs included in the packge. 
These data was originally made available [in the following publication](https://ascpt.onlinelibrary.wiley.com/doi/10.1038/clpt.2014.155)
and obtained from [Physionet](https://physionet.org/content/ecgrdvq/1.0.0/#files-panel).

Each drug has been filtered into its own dataset and is ready to use with `cqtkit`.

<Steps>
1. [`cqtkit_data_verapamil`](../cqtkit_data_verapamil) - C-QT Analysis ready dataset for Verapamil
2. [`cqtkit_data_bl_verapamil`](../cqtkit_data_bl_verapamil) - C-QT Analysis ready pre-study baseline data for Verapamil 
3. [`cqtkit_data_dofetilide`](../cqtkit_data_dofetilide) - C-QT Analysis ready dataset for Dofetilide
4. [`cqtkit_data_bl_dofetilide`](../cqtkit_data_bl_dofetilide) - C-QT Analysis ready pre-study baseline data for Dofetilide
5. [`cqtkit_data_quinidine`](../cqtkit_data_quinidine) - C-QT Analysis ready dataset for Quinidine
6. [`cqtkit_data_bl_quinidine`](../cqtkit_data_bl_quinidine) - C-QT Analysis ready pre-study baseline data for Quinidine
7. [`cqtkit_data_ranolazine`](../cqtkit_data_ranolazine) - C-QT Analysis ready dataset for Ranolazine
8. [`cqtkit_data_bl_ranolazine`](../cqtkit_data_bl_ranolazine) - C-QT Analysis ready pre-study baseline data for Ranolazine
</Steps>
