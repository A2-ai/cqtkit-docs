# `cqtkit` Fit Functions

These functions are focused on generating and working with mixed-effects models for various aspects of C-QTc analyses

<Steps>
1. [`fit_prespecified_model`](../fit_prespecified_model) - This generates the prespecified model from the White Paper on C-QT
1. [`compute_model_fit_parameters`](../compute_model_fit_parameters) - This function extracts the model parameters from the fitted model
1. [`compute_fit_results`](../compute_fit_results) - This function computes the model predictions
2. [`fit_qtc_linear_model`](../fit_qtc_linear_model) - This function fits QTc data to a linear mixed-effect model with (intercept and slope)
</Steps>
