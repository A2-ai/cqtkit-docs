## Model Equation

$$
\begin{aligned}
\Delta \text{QTcF}_{i,j,k} = 
 & \left( \theta_0 + \eta_{0,i} \right ) 
  + \left( \theta_1 + \eta_{1,i} \right ) C_{i,j,k} 
  + \theta_2 \left( \Delta\text{QTcF}_{i,j,k=0} - \text{QTcF}_{\text{baseline}} \right ) \\
 & + \theta_3 \text{TRTG}_j 
  +  \theta_4 \text{TAFD}_k
\end{aligned}
$$
