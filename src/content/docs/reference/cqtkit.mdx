## `cqtkit` Functions

`cqtkit` has seven main classes of functions for analyzing C-QT data:

<Steps>
1. [Preprocessing](../preprocessing/preprocessing) - these functions compute QT corrections and baseline means
2. [EDA](../eda/eda) - these functions generate exploratory data analysis plots
3. [Compute](../compute/compute) - these functions compute various aspects of QT analyses
4. [Tabulate](../tabulate/tabulate) - these functions generate summary and analysis tables 
5. [Fit](../fit/fit) - These functions generate the prespecified linear mixed effects model
6. [GoF](../gof/gof) - these functions generate goodness-of-fit plots of a fitted model
6. [Predict](../predict/predict) - these functions generate prediction plots
</Steps>
