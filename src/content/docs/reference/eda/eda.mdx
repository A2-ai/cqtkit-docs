# `cqtkit` Exploratory Data Analysis Functions 

There are several `eda_` functions that generate various EDA plots for a C-QTc analysis.

<Steps>
1. [`eda_mean_dv_over_time`](../eda_mean_dv_over_time) - This function generates a plot of a dependent variable (dQTc, dHR, Concentration) averaged across all subjects over time.
2. [`eda_qtc_comparison_plot`](../eda_qtc_comparison_plot) - This function generates a plot of the baseline (pre-dose unaveraged QT and RR values) that compares the various corrections used (QTcF, QTcB, QTcP).
3. [`eda_hysteresis_loop_plot`](../eda_hysteresis_loop_plot) - This function generates a plot of averaged dQTc and average Concentration aranged by time.
4. [`eda_scatter_with_regressions`](../eda_scatter_with_regressions) - This function generates a scatter plot of dQTc values plotted against time-matched concentrations overlaid with linear and LOESS regression lines.
5. [`eda_quantiles_plot`](../eda_quantiles_plot) - This function generates a plot of binned dependent variables (Concentration) with mean independent variabl (QTc) at each bin with a linear regression line.
7. [`eda_qt_rr_plot`](../eda_qt_rr_plot) - This function generates a scatter plot of QT vs RR with a linear and LOESS regression line. (It generates a single panel of the eda_qtc_comparison_plot figure)
</Steps>
