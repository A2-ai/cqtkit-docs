# `cqtkit` Preprocessing Functions

<Steps>
1. [`compute_qtcb_qtcf`](../compute_qtcb_qtcf) - This function computes Bazette's and Fridericia's QT correction
1. [`compute_delta_qtcbblm`](../compute_delta_qtcbblm) - This function computes difference between QTcB and mean baseline QTcB
1. [`compute_delta_qtcfblm`](../compute_delta_qtcfblm) - This function computes difference between QTcF and mean baseline QTcF
1. [`compute_delta_hrblm`](../compute_delta_hrblm) - This function computes difference between HR and mean baseline HR
1. [`compute_deltas`](../compute_deltas) - This function comptues delta ECG parameters between baseline and on treatment
1. [`preprocess`](../preprocess) - This is a wrapper function to compute all QT corrections, delta ECG, and delta Baseline Mean values.
</Steps>
