# `cqtkit` Computation Functions

These functions return dataframes of various computations done in C-QT analyses.

<Steps>
1. [`compute_grouped_mean_sd`](../compute_grouped_mean_sd) - This function computes averaged dependent variable grouped by time. 
1. [`compute_pk_parameters`](../compute_pk_parameters) - This function computes pharmacokinetic parameters.
1. [`compute_ecg_param_summary`](../compute_ecg_param_summary) - This function computes summary stastics of ECG parameters.
1. [`compute_high_qtc_sub`](../compute_high_qtc_sub) - This function computes the number of subjects with high QTc observations.
1. [`compute_quantiles_obs_df`](../compute_quantiles_obs_df) - This function computes the average dependent variable within bins of independent variable.
1. [`compute_potential_hysteresis`](../compute_potential_hysteresis) - This function computes if hysteresis is detected within a dose group.
1. [`compute_hysteresis_labeller`](../compute_hysteresis_labeller) - This function computes a labeller function to show potential hysteresis results in facet labels.
1. [`compute_exposure_predictions`](../compute_exposure_predictions) - This function computes predicted effect at drug exposure of interest. 
1. [`compute_dataset_simulation`](../compute_dataset_simulation) - This function computes a simulation of dataset with fitted model.
1. [`compute_summary_statistics_of_simulations`](../compute_summary_statistics_of_simulations) - This function computes the summary statistics of a simulated dataset.
1. [`compute_lm_fit_df`](../compute_lm_fit_df) - This function computes extracts the fitted parameters from a linear regression model.
1. [`compute_enGRI`](../compute_engri) - This function computes the enGRI score for detecting hysteresis.
</Steps>
