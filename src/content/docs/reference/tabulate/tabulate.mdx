# `cqtkit` Table Functions
These functions generate gt tables of various analyses.

<Steps>
1. [tabulate_ecg_param_summary](../tabulate_ecg_param_summary) - This function generates central tendency tables for a ECG parameter (QTc, HR).
1. [tabulate_high_qtc_sub](../tabulate_high_qtc_sub) - This function generates a table of the number of subjects with high QTc values.
1. [tabulate_pk_parameters](../tabulate_pk_parameters) - This function generates a table of Pharmacokinetic parameters ($C_{\text{max}}$, $T_{\text{max}}$).
1. [tabulate_model_fit_parameters](../tabulate_model_fit_parameters) - This function generates a table of estimated fixed effect parameters with confidence intervals. 
1. [tabulate_exposure_predictions](../tabulate_exposure_predictions) - This function generates a table of predicted dependent variable (($\Delta$)$\Delta$QTc, ($\Delta$)$\Delta$HR) at drug exposure levels of interest.
</Steps>
